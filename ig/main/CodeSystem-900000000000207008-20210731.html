Properties

This code system defines the following properties for its concepts

CodeTypeDescription
inactivebooleanWhether the code is active or not (defaults to false).
sufficientlyDefinedbooleanTrue if the description logic definition of the concept includes sufficient conditions (i.e., if the concept is not primitive).
parentcodeThe SNOMED CT concept id that is a direct parent of the concept.
childcodeThe SNOMED CT concept id that is a direct child of the concept.
moduleIdcodeThe SNOMED CT concept id of the module that the concept belongs to.
